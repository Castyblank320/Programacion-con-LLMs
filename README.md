## Repositorio de preguntas y respuestas

Este repositorio contiene las entregas correspondientes a las fases 1 y 2 del ejercicio de Programación con LLMs.

### Estructura

- `myquestions/`: Contiene los archivos de preguntas (`question-XXXX.txt`) y sus generadores de casos de uso (`question-XXXX-usecase-generator.py`).
- `myanswers/`: (Fase 2) Contendrá las soluciones a las preguntas asignadas.

### Preguntas (Fase 1)

1. `question-0001.txt`: Análisis de cohorte de compras.
2. `question-0002.txt`: Optimización de Random Forest con GridSearchCV.
3. `question-0003.txt`: Detección de outliers en producción por máquina.
4. `question-0004.txt`: Pipeline de PCA con regresión logística.

Cada pregunta incluye su respectivo generador de casos de uso.
